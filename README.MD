# My Portfolio

This is a React-based portfolio website built using Vite. It includes pages for About, Portfolio, Resume, and Contact.

1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/your-repo-name.git
   cd your-repo-name

2. ``` npm create vite ```
